Build the Pascal runtime library (pr24p) in Pascal, dogfooding the p24c compiler. Phase 0: hand-written .spc stubs for write_int, write_bool, write_ln. Phase 1: Pascal-compiled standard functions (abs, odd, ord, chr, succ, pred, sqr), bounds/nil checks, and read/readln support. Phase 2: heap management (new/dispose), sets, strings, subrange checks — all in Pascal. Runtime is compiled by p24c into .spc, assembled by pasm into .p24, and runs on pv24a VM (cor24-run emulator or COR24-TD hardware).
